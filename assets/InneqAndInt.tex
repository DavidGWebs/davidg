\section*{Does the Over or Under Hit?}
In order to find areas between curves, we need to find which function is the top, and which is the bottom. The following exercises help you with some ideas to find which is which without graphing.
\begin{enumerate}
    \item Prove the following statements WITHOUT graphing: Let $f(x) = x, g(x) = \sin(x)$
    \begin{enumerate}
        \item For all $x \in [-\frac{\pi}{2}, 0), g(x) > f(x)$.
        \item For all $x \in (0, \frac{\pi}{2}], f(x) > g(x)$.
    \end{enumerate}
    \vfill
    \item Over the interval $(-\pi, \pi)$, determine the intervals where $\cot(x) > \cos(x)$.
    \vfill
    \item On which interval(s) is $\sqrt{|x|}>|x|$?
    \vfill
\end{enumerate}
