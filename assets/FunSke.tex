\section*{Functions that are Fun to Sketch}
As accurately as you can make a sketch of the following three functions:
\begin{enumerate}
    \item $f(x) = \sin(x)+x$ (HINT: What transformation can be applied to $\sin(x)$ to obtain $f(x)$?)
    \vfill
    \item $g(x) = (x^2+1)^{-2/3}+(x^2+1)^{2/3}$
    \vfill
    \item Use your Algebra 2 skills to graph $h(x) = x^4-8x^2+16$, then use Calculus to make sure everything makes sense.
    \vfill
\end{enumerate}

\section*{Function Sketching, but Backwards}
For each of the following functions, sketch a graph which has that function as it's derivative. (This will be cool to know in Week 12-ish).
\begin{enumerate}
    \item $f(x) = |x-2|$
    \vfill
    \item $f(x) = |x|^{1/3}$
    \vfill
    \item $f(x) = \begin{cases}
        \vdots & \vdots\\
        -1 & -1 \leq x < 0\\
        0 & 0 \leq x < 1\\
        1 & 1 \leq x < 2\\
        \vdots & \vdots
    \end{cases}$
    \vfill
\end{enumerate}
